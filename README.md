# ThunderCode Laravel Starter Kit

**Complete Laravel Starter Kit for small business websites.**

## Features

- Admin CRUD for Services (add/edit/delete with image upload)
- Contact form with database storage and optional email notifications
- Frontend pages: Home, Services, About, Contact
- Clean structure, fully functional, minimal styling
- Flexible design – you can change colors, fonts, layout

## Installation

1. Clone the repository: `git clone [(https://github.com/AnnaMmcc/thundercode-starter)]`
2. Enter project folder: `cd thundercode-starter`
3. Install dependencies: `composer install`
4. Copy `.env.example` to `.env` and set your database credentials
5. Run migrations: `php artisan migrate`
6. Create storage link for images: `php artisan storage:link`
7. Serve the app: `php artisan serve`
8. Open in browser: `http://127.0.0.1:8000`

## Admin Panel

- URL: `/admin/services`
- Full CRUD for services with image upload
